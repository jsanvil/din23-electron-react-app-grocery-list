# 2DAM (semi) 2023 DIN: grocery-list-app

Aplicación **_Electron_** con **_React_** y **_Vite_**.

## Configuración del proyecto

### Instalación de dependencias

```bash
$ npm install
```

### Trabajar en desarrollo

```bash
$ npm run dev
```

### Generar ejecutables

```bash
$ npm run build:win

$ npm run build:mac

$ npm run build:linux
```
